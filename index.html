Asystent Tobias

Asystent Tobias is a discord bot for your discord server. If you want to invite bot click "Invite Bot" on top of this page.

Asystent Tobias can help you make the server alive, by lots of commands.

You can check the newest updates here.

Command List

    Info

  • .help - Shows you list of commands...
  • .invite - Gives you link to invite Tobias.
  • .www - Gives you link to this website.
  • .user - Display basic information about user.

    Fun

  • .rate - Rates everything!
  • .% - Rates your personality.
  • .guess - Simple guess game.
  • .year / .bcyear - Century Calculator.
  • .joke - Random joke.
  • .ai - Question to "AI".
  • .rng - Random numbers.

    Moderation

  • .clear - Purge messages.
  • .kick - Kick user.
  • .ban - Ban user.
  • .dg - Check bot health.

    Text

  • .poll - Create poll with custom anwsers.
  • .fp - Create poll "YES/NO".
  • .say - Says everything.

How to invite

It's simple, just click "Invite Bot" button on top of this page or here, if you don't want to give Tobias admin permissions, click "Basic Permissions" button here.
