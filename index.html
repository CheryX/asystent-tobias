Asystent Tobias

Asystent Tobias is a discord bot for your discord server. If you want to invite bot click "Invite Bot" on top of this page.

Asystent Tobias can help you make the server alive, by lots of commands.

You can check the newest updates here.

Command List

    Information

  • .help - Displays help menu.
  • .pn - Displays lastest patch notes.
  • .invite - Invite link command
  • .www - Link to this website

    Fun

  • .rate (argument) - Rates something.
  • .% (argument) - Rates your personality.
  • .zgadnij (cyfra) - Guessing numberes.
  • .rok / .pnerok - Century Calculator.
  • .zart - Random jokes.
  • .funfact - Random fact.
  • .8ball - Question to magic ball.

    Moderation

  • .clear - Purge messages.
  • .kick - Kick user.
  • .ban - Ban user.
  • .dg - Check bot health.
  • .poll - Create poll [YES/NO].

How to invite

It's simple, just click "Invite Bot" button on top of this page or here, if you don't want to give Tobias admin permissions, click "Basic Permissions" button on top of this page or here.
